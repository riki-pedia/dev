ft-to-inv

A tool for converting FreeTube data to Invidious JSON data.

GitHub repository: https://github.com/riki-pedia/ft-to-inv

FreeTube: https://github.com/FreeTubeApp/FreeTube

Invidious: https://github.com/iv-org/invidious

Converts your FreeTube subscriptions, history, and playlists to Invidious JSON format.

Useful for migrating your data between platforms, or for backing it up.

Key Features

  • Convert FreeTube data to Invidious JSON format
  • Sync your data between FreeTube and Invidious
  • Easy to use and lightweight
  • Supports batch processing for large libraries
  • Works anywhere you can get Node.js
  • You can pass in a config file to customize the behavior
  • Open source and actively maintained
